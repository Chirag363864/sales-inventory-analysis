# Sales & Inventory Analytics Dashboard

## 📊 Overview
Developed a data analytics solution to analyze sales performance, product trends, and inventory insights.

## 🛠 Tools Used
- SQL
- Power BI
- Python (Pandas)
- Excel

## 🔍 Key Insights
- Top products contribute major revenue
- West region generates highest sales
- Seasonal sales trends observed
- Identified low-stock items

## 📌 Features
- Sales performance analysis
- Product-wise insights
- Inventory monitoring
